# Graphane Public Website

## Description

This is de documentation about Graphane Data Visualization Microframework.

## Installation

Please, run this command before run the project:

```bash
npm install
```

## Usage

You can run the website with:

```bash
npm run docs:dev
```

This command runs VitePress.
