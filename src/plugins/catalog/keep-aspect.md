# Keep Aspect - plugin

## Load

To load the plugin from the `g-composer` component we must use the URL:

- ```http://localhost:63342/graphane/src/plugins/keep.aspect.js```

```html
<g-composer>
  <script type="plugin"
          src="http://localhost:63342/graphane/src/plugins/keep.aspect.js">
  </script>
</g-composer>
```
